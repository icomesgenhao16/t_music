## t_music

### 运行
切换至项目所在位置，安装依赖
```
npm install
```
运行项目
```
npm run serve
```
接口也需开启，切换至接口所在位置
```
node app.js
```

### 项目说明
[API地址](https://github.com/Binaryify/NeteaseCloudMusicApi)
项目中使用了`element-ui+vuex+vue-router`
实现功能：登录、获取歌单、播放暂停，更多功能待完善
